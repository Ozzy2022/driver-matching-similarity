\# 🚚 Driver Matching System using Similarity



This project simulates a logistics system that matches drivers to transport jobs using similarity models.



\## Features

\- Cosine similarity

\- Haversine distance calculation

\- Business constraints (capacity filtering)

\- Feature weighting



\## Technologies

\- Python

\- Pandas

\- Scikit-learn



\## How to run

```bash

pip install -r requirements.txt

python main.py
